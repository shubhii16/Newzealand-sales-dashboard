New Zealand Sales Analytics Dashboard – Power BI Project

📌 Project Overview

This project is a business intelligence dashboard developed using New Zealand sales dataset to explore overall business performance and regional trends.

The goal is not just visualization, but to convert raw sales data into meaningful insights that support strategic decision-making for business stakeholders.

The dashboard focuses on performance tracking, regional comparison, and time-based sales behavior analysis using interactive visuals.

🎯 Business Goals

Measure overall business performance using key performance indicators

Identify high and low performing regions across New Zealand

Study sales behavior across different product categories

Analyze revenue trends over time for better forecasting

Understand contribution of each segment to total business growth

Build an interactive dashboard for non-technical users

📊 Key Performance Indicators (KPIs)

Total Revenue

Total Profit

Average Order Value

Total Units Sold

Profitability Ratio

🔎 Key Insights from Dashboard

Revenue distribution across different regions and cities

Category-wise contribution to total sales performance

Identification of peak and low-performing regions

Time-series analysis showing sales fluctuation patterns

Comparison of profit vs sales across segments

Interactive filtering using slicers for deeper exploration

🛠️ Tools & Technologies Used

Microsoft Power BI – Dashboard development and visualization

Power Query – Data cleaning and transformation

DAX (Data Analysis Expressions) – KPI calculations and measures

Data Modeling – Relationship building between tables

Excel / CSV – Data source format

Business Intelligence Techniques – Insight generation

⚙️ Project Execution Steps

Imported raw sales dataset into Power BI

Cleaned and transformed data using Power Query

Built data model with proper relationships

Created DAX measures for KPIs

Designed interactive dashboard with visuals and filters

Derived actionable insights from analysis
